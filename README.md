# E-Library Pro

A premium, highly polished, dark-themed full-stack Electronic Library Management Web Application with a 3D glassmorphism interface, interactive backgrounds, and production-ready architecture.

## 🚀 Stack

*   **Frontend**: React (Vite) + Framer Motion + Tailwind CSS + Three.js + Recharts
*   **Backend**: Node.js + Express + Mongoose (MongoDB Atlas or Local fallback)
*   **Validation**: React Hook Form + Zod Schema Validation
*   **Icons & Design**: Lucide React + Glassmorphic elements

## 📂 Architecture

The project is split into standard folders designed for scalability:
- `/client` holds the React Single Page Application config, assets, contexts, hooks, components, and views.
- `/server` holds the Node/Express backend models, routes, controller business logic, and seed data scripts.

## ⚙️ Key Features

- **Dynamic Interactive 3D Background**: Features custom Three.js responsive star fields.
- **Glassmorphic Interactive UI**: Fully customizable responsive dashboards, neon accents, and custom pointer hover-tilt card effects.
- **Robust Book & Member Catalog**: Real-time CRUD and state synchronizations.
- **Comprehensive Borrow-Return Management**: Multi-step business rules handling active borrows, returns, overdue updates, and fine calculations.
